# My-Doc — Centralised courses & technical tests

Ce dépôt regroupe tous mes cours, notes et anciens tests techniques en un seul endroit.

Objectifs:

- Rassembler les fichiers Markdown (cours, cheat-sheets, tests techniques).
- Fournir un index généré automatiquement pour naviguer rapidement.
- Offrir des commandes et scripts utiles pour maintenir le dépôt.

Utilisation rapide:

1. Rendre le script exécutable: `chmod +x scripts/generate_index.sh`
2. Générer l'index: `./scripts/generate_index.sh`
3. Ouvrir `INDEX.md` pour parcourir le contenu.

Voir `Commands.md` pour les commandes courantes.
